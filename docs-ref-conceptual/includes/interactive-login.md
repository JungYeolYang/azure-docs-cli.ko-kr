1. 로그인 명령을 실행합니다.

    ```azurecli-interactive
    az login
    ```

   다음 단계에 사용할 코드를 가져옵니다.

2. 웹 브라우저를 사용하여 [https://aka.ms/devicelogin](https://aka.ms/devicelogin) 페이지를 열고 제공된 코드를 입력하여 인증하세요.

3. 브라우저에서 계정 자격 증명으로 로그인합니다.
